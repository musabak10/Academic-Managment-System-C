Student Academic Management System

A menu-driven Student Academic Management System developed in C. The project manages students, courses, grades, and academic records using structures, pointers, and dynamic memory allocation.

Features
Add multiple students
Add and delete courses
Enter midterm and final grades
Calculate weighted course averages
Calculate overall GPA based on course credits
Update course information
Search courses by course code
Find the highest graded course
List failed courses
Print a detailed report card
Display total credit load
Sort courses alphabetically
Update student information
Copy courses from one student to another
